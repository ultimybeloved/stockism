Firebase Console Configuration Steps

To fix the 'auth/internal-error' when signing in with Google on localhost:

1. Go to Firebase Console: https://console.firebase.google.com/project/stockism-abb28
2. Navigate to Authentication > Settings > Authorized domains
3. Make sure 'localhost' is in the authorized domains list
4. If not, click 'Add domain' and add 'localhost'

5. Also go to Google Cloud Console:
   https://console.cloud.google.com/apis/credentials?project=stockism-abb28
6. Find your OAuth 2.0 Client ID for Web application
7. Under 'Authorized redirect URIs', add:
   - http://localhost:5174/__/auth/handler
   - http://localhost:5173/__/auth/handler
8. Click Save
